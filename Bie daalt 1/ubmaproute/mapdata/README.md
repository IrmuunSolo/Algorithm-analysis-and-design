# mapdata

Place Geofabrik Mongolia roads shapefile here before running the app.

Required files (same basename):
- `gis_osm_roads_free_1.shp`
- `gis_osm_roads_free_1.dbf`
- `gis_osm_roads_free_1.shx`
- `gis_osm_roads_free_1.prj`
- `gis_osm_roads_free_1.qix` (optional but recommended)

Source: https://www.geofabrik.de/data/download.html (Asia → Mongolia → roads)

The repository `.gitignore` excludes large datasets by default. Commit them only if necessary, ideally via Git LFS.
